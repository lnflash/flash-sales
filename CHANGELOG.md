# Flash Sales Dashboard - Changelog

## v1.1.0 - April 16, 2025

### Features
- Added submission edit functionality
- Created edit page for submissions at /dashboard/submissions/[id]/edit
- Updated API client to support updating submissions
- Enhanced React Query hooks with mutation capabilities

### Improvements
- Fixed 404 error when clicking "Edit" on submission details
- Enhanced API proxy to handle PUT requests
- Improved error handling throughout the application
- Added loading states to edit form

## v1.0.0 - April 12, 2025

### Features
- Initial release of the Flash Sales Dashboard
- Dashboard overview with key metrics and charts
- Submission listing with filtering and sorting
- Submission detail view
- API integration with Flash Intake Form
- Responsive design for desktop and mobile
- Dark theme with Flash branding
